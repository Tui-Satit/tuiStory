ยินดีต้อนรับทุกท่าน

สวัสดีทุกท่าน ผมสร้างเว็บไซด์นี้ขึ้นมาเพื่อนำเสนอความรู้สึกนึกคิด บันทึกความทรงจำ

Storytelling: เที่ยวญี่ปุ่น

เมื่อปลายเดือนกุมภาพันธ์-ต้นเดือนมีนาคม (26 ก.พ. 67-6 มี.ค.67) ผม พี่สาวและเพื่อนพี่สาวอีก 1 คนไปเที่ยวญี่ปุ่นกัน 9 วัน ออกจากสนามบินสุวรรณภูมิตอนเที่ยงคืนถึงญี่ปุ่นที่สนามบินนาริตะตอน 7 ตอนโมงเช้าจากนั้นนั่งรถบัสไปจังหวัดคาวากุจิโกะ ใช้เวลาเดินทางประมาณ2ชั่วโมงที่พักที่คาวากุจิโกะอยู่เชิงเขาคืนวันแรกไปกินอาหารที่ร้านปิ้งย่างถูกใจมาก

วันที่2ขึ้นกระเช้า ไปดูภูเขาไฟฟูจิและไปเที่ยวชมหมู่บ้านแม่น้ำ วันที่ 3 เดินทางเข้าโตเกียวพักที่ย่านกินซ่าใกล้ตลาดปลาสึกิจิ วันที่4 เดินชม ตลาดปลาสึกิจิ ไปเทียวที่ชินจุกุ ไปหารองเท้า On running ที่ฮาราจุกุ วันที่5 ไปเที่ยววัดอาซากุสะ (วัดโคมแดง)และ นั่งรถไฟไปช็อปปิ้งที่อุอิโนะ ซื้อรองเท้า On running ที่ห้าง Pargo นั่งรถไฟไปที่ชิบุญ่าถ่ายรูปกับ ฮาจิโกะ สุนัขผู้ซื่อสัตย์ เดินข้ามแยกชิบุญ่าเป็นประสบการณ์ชีวิต วันที่6 นั่งรถไฟชินกันเซนมาที่โอซาก้า ช่วงเย็นไปเที่ยวย่านโดทงโบริ ถ่ายรูป กับกุลิโกะ วันที่7 มาเที่ยวย่านโดทงโบริอีกครั้ง ช็อปปิ้งที่ย่านเซนไชบาชิ วันที่ 8 ไปเที่ยวพิพิธภัณฑ์โอซาก้า และ พิพิธภัณฑ์สัตว์น้ำ นั่งรถไฟมากินอาหารค่ำที่โดทงโบริ วันที่9 เตรียมตัวกลับกรุงเทพ ตอนเช้าไปแลกเงินที่ย่านเซนไชบาชิ กินบิงซูที่โดทงโบริ นั่งรถไฟเข้ามาที่สนามบินเคนไซ
สรุป การได้ไปเที่ยวญี่ปุ่นเป็นประสบการณ์ที่ดีมาก คนญี่ปุ่นมีอัธยาศัยดีมาก บ้านเมืองสะอาด มีระเบียบ อากาศหนาวเย็น กำลังดี การเดินทางใช้รถไฟเป็นส่วนใหญ่ เจอนักท่องเที่ยวไทยเยอะมากเจอทุกที่

ศุกร์ 25,10,67

วันนี้ตอนเช้าเข้าร่วมอธิฐานเช้าออนไลน์กับกลุ่ม M02 กินกาแฟ อ่านมานา กินอาหารเช้าเสร็จ ก็เรียน ภาษาอังกฤษกับ BBC จากนั้นก็ศึกษาโปรแกรม Python บ่ายก็เขียนเว็บจนค่ำ

พฤหัส 24.10.67

วันนี้ตื่นสายไม่ได้ออกกำลังกาย ตอนเช้ากินกาแฟกับแพนเค้ก อ่านมานาประจำวัน เรียนภาษาอังกฤษกับ BBC กินอาหารเช้าเสร็จ เริ่มเรียนโปรแกรม Python เป็นภาษาเขียน Program ตอนบ่ายกินข้าวเที่ยงเสร็จ เริ่ม อัพเดทเว็บไซด์ เว็บนี้เป็นเว็บทดลอง เขียนไปเป็นแบบลองผิดลองถูก #คุยไปเรื่อย
